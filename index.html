Featured Fundamentals
Variable

A variable is a storage location that holds a value, which can be changed during program execution.

Input and Output

Input refers to data provided by the user or external sources. Output displays the results of computations or actions.

Conditional Statements

Conditional statements help a program make decisions based on specific conditions being true or false.

Looping Statements

Loops are used to repeat tasks multiple times based on certain conditions, reducing repetitive code.
